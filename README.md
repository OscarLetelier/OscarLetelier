## KuraGym — Proyecto principal

Plataforma SaaS Multi-tenant para gestión de gimnasios, construida desde cero con **Domain-Driven Design**: dominio modelado en agregados y bounded contexts para garantizar escalabilidad y mantenibilidad a largo plazo.

```
Backend   →  Java 21 · Spring Boot 3 · Spring Security · JPA · MapStruct
Frontend  →  React · TypeScript · Tailwind CSS · Vite
DevOps    →  Docker · SonarQube · JUnit 5 · Vitest
Patrón    →  DDD · API-First · Multi-tenant · SOLID
```
